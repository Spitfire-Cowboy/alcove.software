# FAQ

## Is Alcove an LLM chatbot?

No. Alcove is retrieval infrastructure. It finds indexed source text and returns it with provenance.

## Does Alcove require cloud APIs?

No. Core ingest/index/query flows are local-first. Operators control deployment boundaries.

## Is AI required?

No. AI-assisted semantic search is optional.

## What is in this repository?

Public site assets and ecosystem documentation for `alcove.software`.

## Where is the core engine?
