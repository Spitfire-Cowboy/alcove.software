# Install

Alcove core install is handled in the main engine repository:

- https://github.com/Spitfire-Cowboy/alcove

Recommended path for operators:

1. Install Python 3.10+.
2. Install package with semantic retrieval extras.
3. Run local demo + serve commands.
4. Validate local ingest/index/query loop before production data.

This public site repo does not contain the Alcove engine runtime. It hosts public docs, crawler policy files, and ecosystem handoff information.
