Alcove, indexed for the open web.

This site is the public surface for Alcove: local-first search infrastructure. Alcove retrieves real document text from local indexes. It is retrieval, not generation.

Local-first by design

Data stays on operator-controlled disk by default. No mandatory cloud dependencies.

Retrieval system

Alcove finds relevant chunks and returns them. It does not fabricate new content.

AI optional

Semantic embedding is opt-in. Operators choose whether and where AI is involved.
